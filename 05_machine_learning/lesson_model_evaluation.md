# レッスン: モデル評価入門

概要:
分類モデルの性能指標（精度、再現率、適合率、F1、混同行列、ROC-AUC）を理解し、scikit-learn を使って実際に計算・解釈できるようにします。

難易度: 初級〜中級
所要時間: 30–45分
前提: scikit-learn の基本的な使い方、モデルの学習ができること

学習項目:
- 混同行列の見方と計算方法
- 精度（accuracy）、適合率（precision）、再現率（recall）、F1 スコア
- ROC曲線と AUC の役割
- 分類閾値の調整が与える影響

演習:
- サンプル分類器の予測で各指標を計算する
- 閾値を変えて適合率／再現率のトレードオフを観察する

実行: model_evaluation_example.py を実行して出力を確認してください。
