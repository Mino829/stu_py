# レッスン: 前処理入門

概要:
機械学習モデルを安定して学習させるための基本的なデータ前処理（欠損値処理、スケーリング、カテゴリ変換）を学びます。初心者向け、ハンズオン中心。

難易度: 初級
所要時間: 30–45分
前提: Pythonの基本（変数、関数）、pandasの簡単な操作がわかること

学習目標:
- 欠損値の検出と基本的な処理方法を説明できる
- 数値特徴量のスケーリング方法を実装できる
- カテゴリ変数を数値に変換できる

ステップ:
1. データの読み込み（今回はサンプルデータを使用）
2. 欠損値の確認と処理（drop / fill）
3. カテゴリ変数の変換（get_dummies / LabelEncoder）
4. スケーリング（StandardScaler）
5. 保存と再利用（簡単なserialize例）

演習:
- 与えられたCSVで欠損値の割合が高い列を削除する
- get_dummies と LabelEncoder の違いを説明する

参考: preprocessing_example.py を実行して動作を確認してください。
