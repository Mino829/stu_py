# レッスン: データ可視化入門

概要:
機械学習での探索的データ解析（EDA）でよく使う可視化手法の基礎を学びます。Matplotlib と pandas を使った実践的な例で理解を深めます。

難易度: 初級
所要時間: 30分
前提: pandas の基本操作、Python の基礎

学習項目:
- 単変量のヒストグラム・箱ひげ図
- 二変量の散布図と相関の確認
- カテゴリ毎の棒グラフ

演習:
- サンプルデータを可視化して特徴量の分布を報告する
- 散布図に回帰線を追加して関係を観察する

実行: visualization_example.py を実行して図を表示してください。
