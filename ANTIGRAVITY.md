# Project Instructions for Antigravity CLI

このプロジェクトは、Python初心者が機械学習エンジニアへの第一歩を踏み出すための環境です。

## アシスタントへの指示
- **インタラクティブな指導**: ユーザーに質問を投げかけたり、小さな課題（クイズ）を出したりして、能動的な学習を促してください。
- **可視化の推奨**: `04_data_libraries` 以降では、常にデータを可視化して理解することを推奨してください。
- **簡潔なコード**: 初学者が読みやすい、シンプルで標準的なコード（PEP 8準拠）を提示してください。

## 学習のゴール
- 自分でデータセットを読み込み、簡単な予測モデルを構築して評価できること。

## 構成
各ディレクトリの `exercise.py` を埋めていく形式で学習を進めることを想定しています。
