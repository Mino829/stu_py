# レッスン: 特徴量エンジニアリング入門

概要:
モデル性能向上に直結する特徴量エンジニアリングの基礎を学びます。特徴量の生成、選択、変換の実践的手法をハンズオンで確認します。

難易度: 初級〜中級
所要時間: 45分
前提: pandas, NumPy, scikit-learn の基本操作

学習項目:
- 新しい特徴量の作成（組合せ・算術変換）
- 多項式特徴量（PolynomialFeatures）の利用
- 特徴量選択（SelectKBest, 重要度に基づく選択）
- カテゴリ特徴量の取り扱いと交互作用

ステップ:
1. サンプルデータを読み込む
2. 単純な演算で特徴量を作る（例: age*salary）
3. PolynomialFeatures で高次特徴量を生成
4. SelectKBest とランダムフォレストの重要度で重要な特徴量を選ぶ
5. 選択した特徴量で簡単なモデルを学習し比較する

演習:
- 多項式次数を増やして過学習の傾向を観察する
- 特徴量選択の結果を README に書いて説明する

実行: feature_engineering_example.py を実行して出力を確認してください。
